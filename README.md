 # Student Management System

A simple Student Management System built using C language.
This project uses file handling to store and manage student records.

---

## 🎯 Features
- Add new student
- Display all students
- Search student by ID
- Delete student record

---

## 🛠️ Tech Stack
- C Language
- File Handling

---

## ⚙️ How to Run

1. Clone the repository:
   git clone https://github.com/jayadritojana2006-debug/student-management-system.git

2. Navigate to the folder:
   cd student-management-system

3. Compile the code:
   gcc main.c

4. Run the program:
   ./a.out
